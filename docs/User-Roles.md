# User Roles

## System Administrator

- Manage users
- Manage roles
- Configure system
- View reports
- Manage departments

---

## Asset Manager

- Add assets
- Update assets
- Assign assets
- Track maintenance
- Generate reports

---

## Employee

- View assigned assets
- Request maintenance
- Report damaged assets

---

## Auditor

- Read-only access
- Generate compliance reports
- View audit history
